# 🚀 Meu Portfólio Pessoal

Este é o meu projeto de portfólio pessoal, desenvolvido para centralizar meus projetos, habilidades e experiências como Desenvolvedor Full Stack buscando sua primeira oportunidade de Estágio. O projeto foi construído utilizando as tecnologias mais modernas do ecossistema React para garantir performance, SEO e uma excelente experiência de usuário.
## 🛠️ Tecnologias Utilizadas
* Framework: Next.js 15 
* Linguagem: TypeScript
* Estilização: CSS Modules 
* Animações: ReactBits e CSS
* Ícones: React Icons (Font Awesome)
## 📂 Estrutura do Projeto
A arquitetura foi pensada para ser escalável e organizada:
* src/app/: Contém os Arquivos principais da aplicação como o Page.tsx e layout.tsx.
* src/components/: Componentes reutilizáveis da interface (Hero, Projetos, Skills, etc.).
* src/app/styles/: Arquivos de estilização  e módulos específicos.
* src/app/globals: Arquivo de estilização Global.
* public/: Assets estáticos como imagens e logotipos.
## 🎨 Funcionalidades Principais
* Design Responsivo: Adaptado para dispositivos móveis e desktops.
* Componentes do Lado do Cliente (Client Components): Utilização de Hooks (useState, useEffect) para interatividade, como o carrossel de logos (LogoLoop).
* SEO Optmized: Configurado com os metadados nativos do Next.js para melhor ranqueamento.
* Seções Inclusas:
* Hero: Apresentação principal impactante.
* Skills: Exibição técnica de habilidades.
* Projetos: Galeria de trabalhos realizados.
* Contato: Links diretos para redes sociais (GitHub, LinkedIn).
## ⚙️ Como executar o projeto

### Clone o Repositório
* No terminal: git clone https://github.com/seu-usuario/portfolio-dev-lucas.git

### Instale as Dependências
* No terminal: npx jsrepo@latest add https://reactbits.dev/r/LogoLoop-TS-CSS e install npm install react-icons
* No terminal: npx create-next-app@latest(Para criar a estrutura do projeto utilizando next)

### Inicie o Servidor
* No teminal: npm run dev

### Acesse o Navegador
* http://localhost:3000
